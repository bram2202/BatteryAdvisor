# Battery Advisor

## Backend

An C# .net core 10 backend with REST API.

How to start:
```
cd BatteryAdvisor.Api
dotnet run
```

How to build
```
dotnet build BatteryAdvisor.slnx
```

How to test
```
dotnet test BatteryAdvisor.slnx
```


## Frontend

An Angular 21 application using TailwindCSS and PrimeNG

How to start:
```
corepack pnpm start
```
